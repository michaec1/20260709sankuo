# 人物

- [[劉備]] — 蜀漢建立者，從流亡軍事領袖成為益州、漢中之主，最後在成都稱帝。
- [[關羽]] — 劉備早期核心將領，曾斬顏良、督守荊州，北伐樊城後敗亡。
- [[張飛]] — 劉備重要將領，長坂斷後、入蜀立功，後在伐吳前被部下所殺。
- [[馬超]] — 關中、涼州出身的武將，後投奔劉備，促成成都震動與劉璋投降。
- [[黃忠]] — 蜀漢將領，在定軍山斬夏侯淵，是漢中之戰的重要功臣。
- [[趙雲]] — 劉備麾下忠勇將領，長坂保護劉禪，入蜀與北伐皆有參與。
- [[諸葛亮]] — 蜀漢丞相，提出隆中戰略，促成孫劉聯盟，受託輔政並北伐曹魏。
- [[曹操]] — 曹魏奠基者，統一北方，官渡勝袁紹，赤壁敗於孫劉聯軍。
- [[董卓]] — 東漢末年軍閥，入京掌權、廢立皇帝，直接引爆群雄割據。
- [[袁紹]] — 北方大軍閥，出身汝南袁氏，官渡敗於曹操後勢力衰退。
- [[袁術]] — 淮南割據者，曾僭號稱帝，因失政與眾叛親離而敗亡。
- [[劉表]] — 荊州牧，長期據有荊州，死後劉琮降曹，引發劉備南撤。
- [[孫權]] — 東吳建立者，接掌江東後抗曹、奪荊州，維持三國鼎立局勢。
- [[程普]] — 東吳宿將，歷仕孫堅、孫策、孫權，在赤壁與周瑜同為左右督。
- [[黃蓋]] — 東吳將領，赤壁之戰提出並執行火攻，是破曹的重要人物。
- [[韓當]] — 東吳早期武將，擅長弓馬，參與赤壁、南郡與夷陵等戰事。
- [[蔣欽]] — 東吳將領，長於水軍與地方討賊，合肥之戰與討關羽時皆有參與。
- [[周泰]] — 東吳勇將，曾在宣城捨身保護孫權，後督守濡須。
- [[陳武]] — 東吳將領，受孫權親近，215 年合肥之戰中奮戰而死。
- [[董襲]] — 東吳將領，討黃祖時斷敵船繫索，後在濡須防守中遇風船敗身死。
- [[甘寧]] — 東吳名將，投孫權後參與破黃祖、赤壁、益陽與濡須等戰事。
- [[凌統]] — 東吳將領，合肥撤軍時率親兵護衛孫權突圍。
- [[徐盛]] — 東吳將領，以勇氣聞名，曾建議設疑城阻止魏文帝渡江。
- [[潘璋]] — 東吳將領，軍令嚴整，荊州之爭中其部下擒獲關羽。
- [[丁奉]] — 東吳後期將領，東興之戰突擊魏軍，後參與誅殺孫綝。
- [[周瑜]] — 東吳名將，主導赤壁抗曹並以火攻擊敗曹操。
- [[魯肅]] — 東吳謀臣，主張聯劉抗曹，並處理赤壁後的荊州交涉。
- [[呂蒙]] — 東吳將領，由勇將轉為智將，白衣渡江襲取荊州。
- [[董和]] — 蜀漢官員，以清廉節儉與直言輔政著稱，曾與諸葛亮共署府事。
- [[劉巴]] — 蜀漢文臣，曾北投曹操，後仕劉備，負責財政與重要文告。
- [[馬良]] — 蜀漢官員與使者，曾出使東吳，夷陵之戰後遇害。
- [[馬謖]] — 蜀漢幕僚，提出攻心為上，但街亭失守後下獄而死。
- [[陳震]] — 蜀漢外交官，孫權稱帝後奉命赴吳盟誓，維繫蜀吳合作。
- [[董允]] — 蜀漢後主時期重臣，匡正宮中政務並約束黃皓。
- [[呂乂]] — 蜀漢官員，歷任太守與尚書令，以清能與嚴法著稱。

# 事件

- [[桃園三結義]] — 小說與民間敘事中的象徵事件，用來理解劉備、關羽、張飛的兄弟情誼。
- [[董卓亂政]] — 189 至 192 年董卓入京掌權、廢立皇帝、遷都長安，使東漢中央崩壞。
- [[官渡之戰]] — 200 年曹操擊敗袁紹，奠定北方優勢的關鍵戰役。
- [[袁術稱帝]] — 197 年袁術在淮南僭號稱帝，失去政治正當性並快速敗亡。
- [[長坂之戰]] — 208 年曹操追擊劉備，張飛斷後、趙雲護主的危急戰役。
- [[赤壁之戰]] — 208 年孫權與劉備聯軍擊敗曹操，奠定三國鼎立格局。
- [[合肥之戰]] — 215 年孫權攻合肥撤軍時遭張遼突襲，東吳諸將力戰護主。
- [[劉備入蜀]] — 211 至 214 年劉備由受邀入益州轉為攻取成都，建立蜀漢根基。
- [[漢中之戰]] — 217 至 219 年劉備擊退曹操勢力，取得漢中並自立為漢中王。
- [[關羽北伐]] — 219 年關羽圍攻樊城，水淹七軍，一度威震華夏。
- [[荊州之爭]] — 赤壁後孫劉雙方爭奪荊州，最後以呂蒙襲取南郡、關羽敗亡告終。
- [[夷陵之戰]] — 221 至 222 年劉備伐吳失利，蜀漢元氣大傷。
- [[街亭之戰]] — 228 年馬謖在諸葛亮北伐中失守街亭，導致蜀軍退回漢中。
- [[東興之戰]] — 252 年丁奉率兵快速突擊魏軍前屯，促成東吳大勝。
