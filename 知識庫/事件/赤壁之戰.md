# 背景

[[曹操]]取得荊州後，順江東下，想進一步壓服江東。當時[[劉備]]在[[長坂之戰]]後退到夏口附近，處境危急；[[諸葛亮]]奉命前往江東說服[[孫權]]抗曹，[[魯肅]]也主張聯劉抗曹。

# 經過

[[周瑜]]判斷曹軍遠來疲憊、不習水戰，又有疫病與補給問題，因此主張迎戰。孫權同意後，派[[周瑜]]、程普等率軍與劉備合力。交戰時，黃蓋以詐降接近曹軍船隊，趁風放火，曹軍船艦與岸上營寨被燒，曹操被迫北退。

# 影響

赤壁之戰阻止了曹操一舉南下統一的局面，使孫權保住江東，劉備也得到喘息與發展空間。此戰之後，魏、蜀、吳三方分立的格局逐漸成形。
