# 背景

[[赤壁之戰]]後，[[孫權]]多次北上攻魏，希望突破淮南防線。合肥是曹魏防守江淮的重要據點，因此成為孫權北伐的重要目標。

# 經過

215 年，孫權攻合肥。撤軍時魏將張遼突襲，東吳諸軍一度混亂。[[陳武]]戰死，[[潘璋]]斬逃兵穩住軍心，[[凌統]]率親近兵護衛孫權突圍，[[甘寧]]等人也參與力戰。

# 影響

合肥之戰顯示東吳北上淮南的困難，也使張遼威名大振。對東吳而言，這場戰役留下多位將領奮戰護主的記錄，成為理解孫權軍事集團的重要事件。
