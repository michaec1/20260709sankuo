# 背景

東漢末年，[[曹操]]與[[袁紹]]爭奪北方主導權。袁紹兵力較盛，並據有冀州等地；曹操則控制許都與漢獻帝，雙方在官渡長期對峙。

# 經過

曹操軍糧緊張時，袁紹謀士許攸投奔曹操，建議襲擊袁紹在烏巢的糧草。曹操親率精兵夜襲烏巢，焚毀袁紹輜重，造成袁軍大亂。張郃、高覽等將領投降，袁紹敗走。

# 影響

官渡之戰是曹操統一北方的關鍵轉折。袁紹敗後勢力衰退，曹操逐步取得冀州與北方優勢，日後才有南下荊州、引發[[赤壁之戰]]的條件。
