# 背景

[[關羽]]敗亡與荊州失守後，[[劉備]]對[[孫權]]極為憤怒。221 年劉備稱帝後，決定東征東吳，為關羽與荊州之失復仇。

# 經過

劉備率軍沿長江東進，吳將陸遜採取持守策略。222 年，劉備軍在夷道猇亭一帶連營，陸遜抓住時機反攻，大破蜀軍。劉備退回白帝城，後改魚復為永安。

# 影響

夷陵之戰使蜀漢元氣大傷，也讓劉備在永安病重。223 年劉備臨終託孤於[[諸葛亮]]，蜀漢進入諸葛亮輔政時期。戰後孫權也因擔心蜀漢報復而重新請和，孫劉關係逐漸修復。
