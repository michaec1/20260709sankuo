# 背景

[[赤壁之戰]]後，荊州成為孫劉雙方都重視的戰略地帶。[[劉備]]需要荊州作為北伐與入蜀的跳板，[[孫權]]則擔心長江上游被蜀漢掌握。

# 經過

劉備取得益州後，孫權要求歸還荊州部分郡縣。雙方一度緊張，[[魯肅]]與[[關羽]]在益陽交涉，最後以湘水為界暫時罷兵。219 年關羽北伐樊城，[[呂蒙]]趁其後方空虛，白衣渡江襲取南郡，糜芳、士仁投降，關羽退走麥城後被俘殺。

# 影響

荊州之爭使孫劉聯盟出現重大裂痕，也直接引發[[劉備]]日後東征東吳，最終導致[[夷陵之戰]]。
