# 背景

曹操取得漢中後，夏侯淵、張郃等人留守。漢中是益州北方門戶，若由曹操控制，[[劉備]]在益州的安全會受到威脅。

# 經過

劉備率軍進攻漢中，與曹軍相持。[[張飛]]曾在宕渠擊敗張郃，穩住巴西。219 年，劉備在定軍山部署，命[[黃忠]]乘高進攻，斬殺夏侯淵。曹操雖親自南征，但劉備據險不戰，曹軍久攻不利而撤退。

# 影響

劉備取得漢中後，自立為漢中王，蜀漢勢力達到高峰。此戰也讓[[曹操]]在西南方向的擴張受阻。
