# 背景

東漢靈帝死後，何進與宦官勢力衝突，朝廷陷入混亂。何進召外兵入京，[[董卓]]因此率軍進入洛陽，並趁中央失序掌握兵權。

# 經過

董卓入京後，廢少帝、立獻帝，並殺害弘農王與何太后。他以相國、太師等身份壓制百官，刑罰嚴酷，又在關東諸州起兵反對時強迫遷都長安，焚毀洛陽宮室、挖掘陵墓，造成極大破壞。

# 影響

董卓亂政使東漢中央威信崩潰，各地州牧與軍閥開始以討董、勤王或自保為名擴張勢力。[[曹操]]、[[袁紹]]、[[袁術]]等人的崛起，都與這場中央崩壞有直接關係。
