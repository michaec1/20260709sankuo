# 背景

益州牧劉璋面對張魯與北方壓力，邀請[[劉備]]入蜀協助。劉備原本駐於荊州，依[[諸葛亮]]隆中戰略，益州正是建立根據地的重要方向。

# 經過

劉備入蜀後，先在葭萌厚樹恩德。後因張松事泄、劉璋與劉備互不信任，雙方決裂。劉備斬白水軍督楊懷，進軍攻取益州；[[黃忠]]、卓膺等隨軍作戰，[[諸葛亮]]、[[張飛]]、[[趙雲]]也從荊州入蜀分定郡縣。214 年成都被圍，劉璋出降。

# 影響

劉備取得益州後，終於擁有穩定根據地，蜀漢政權的基礎形成。此後他能與曹操爭奪漢中，進一步走向三國鼎立。
