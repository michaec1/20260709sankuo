# 背景

曹操南征荊州時，劉表之子劉琮投降。[[劉備]]從樊城南撤，途中有大量荊州百姓跟隨，行軍速度很慢。[[曹操]]擔心劉備先占江陵，因此以精騎急追。

# 經過

曹軍在當陽長坂追上劉備。劉備被迫棄妻子，與[[諸葛亮]]、[[張飛]]、[[趙雲]]等數十騎逃走。張飛斷後據橋喝退追兵，趙雲則保護甘夫人與劉禪脫險。

# 影響

長坂之戰讓劉備勢力遭受重創，但也促使劉備轉向聯合[[孫權]]。此後諸葛亮出使江東，間接推動[[赤壁之戰]]的孫劉聯盟。
